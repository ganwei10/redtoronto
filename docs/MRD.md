## 2. 市场背景

- **平台侧**：小红书月活过亿，内容形态以「种草 / 生活方式 / 本地生活」为主，海外华人（留学生、新移民、陪读家庭）高度依赖小红书获取本地生活信息（买房、移民、医美、餐饮、母婴）。
- **城市侧**：多伦多是北美华人最密集城市之一，华人人口约 70 万，华人商户在留学移民、地产、医美、餐饮、母婴、旅游保险等领域高度集中。
- **供给侧缺口**：小红书广告代理商掌握「聚光（投放）+ 蒲公英（创作者合作）」能力，但**本地 BD、内容本地化、博主组织**是其下沉到具体城市商户的短板——代理商有平台却触达不到多伦多街头商户。

---

## 3. 目标市场与用户

| 角色 | 描述 | 核心诉求 |
|---|---|---|
| 核心客户（B 端） | 多伦多华人商户（6 大优先赛道） | 低门槛做小红书营销、拿到可衡量效果 |
| 间接用户 | 本地博主（KOC/KOL） | 稳定商单、合理报价、少扯皮 |
| 合作伙伴 | 小红书广告代理商 | 下沉获客、扩大投放量 |
| 平台运营（我们） | 多伦多本地化服务商 | 借代理商合规通道，沉淀系统化能力 |

**6 大优先赛道**（华人/新移民强需求、商户有预算、内容易起量）：
1. 留学 / 移民 / 语言培训
2. 地产（买房 / 租房 / 楼花）
3. 医美 / 美容 / 护肤
4. 餐饮（奶茶、火锅、早茶）
5. 母婴 / 月子 / 育儿
6. 旅游 / 保险 / 理财

## 4. 用户痛点

- **商户**：不懂平台规则、缺内容生产能力、找不到合适博主、效果不可衡量、怕踩合规红线。
- **博主**：接单零散、报价混乱、缺乏稳定商单与结算保障。
- **代理商**：本地触角缺失，难以直接服务城市级中小商户。
- **我们（新人）**：资源有限，必须以「系统 + 本地关系」作为杠杆，而非堆人头。

---

## 5. 机会与价值主张

**定位**：做代理商的「多伦多本地化服务商」——商户 BD + 博主组织 + 内容本地化 + 效果归因，统一用一套 AI 系统提效。

| 对象 | 价值主张 |
|---|---|
| 商户 | 一个入口搞定「找博主→出内容→投放→看效果」 |
| 博主 | 稳定单量、清晰报价、交稿即结算 |
| 代理商 | 低成本下沉获客，投放量提升 |
| 我们 | 轻资产、可复制、数据沉淀为壁垒 |

---

## 6. 市场规模（示意估算）

- 多伦多华人 ~70 万；6 大赛道活跃商户数千家。
- 单商户月营销预算 CAD 1k–20k（差异大）。
- 服务费市场（服务商佣金 + 投放差价）年规模估算为 **CAD 数千万级**（示意，需实际调研校准）。
- 早期聚焦 1–2 赛道打透，再横向复制。

---

## 7. 竞争格局

- **直接**：本地华人营销工作室（纯人工、无系统）、国内 MCN 出海团队（缺本地化）。
- **间接**：通用 KOL 平台（如国内蒲公英本身，但无本地 BD 与跨文化）。
- **差异化壁垒**：AI 驱动的**博主匹配 + 内容本地化 + 效果归因**系统化能力，叠加代理商合规投放通道。

---

## 8. 商业模式

- **现阶段**：项目制服务费 + 投放差价（走代理商合规账户）。
- **未来**：SaaS 订阅（博主库 + 看板 + Brief 生成）面向中小商户 / 其他城市代理商输出能力。

---

## 9. 成功指标（北极星）

| 阶段 | 目标 |
|---|---|
| 0–1 月 | 3–5 个 pilot 商户，跑通 1–2 赛道小闭环 |
| 2–3 月 | 沉淀 20–30 个本地博主，积累内容模板与效果数据 |
| 4–6 月 | 系统上线，月活商户 20+，复购率 > 40% |
| 6–12 月 | 引入匹配推荐与 Agent 复盘，横向复制赛道 |

## 10. 风险与缓解

| 风险 | 缓解 |
|---|---|
| 平台政策（蒲公英合规） | 付费合作一律走蒲公英，绝不私单 |
| 加拿大广告/隐私法规 | 合作披露（material connection）、PIPEDA 数据合规 |
| 跨境税务 | 走代理商结算、留票据、不私账走账 |
| 博主供给波动 | 签/绑定腰部 KOC，稳定单量换忠诚 |
